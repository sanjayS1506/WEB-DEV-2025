4.1 Introduction to Links Definition Links in HTML, created using the tag (anchor tag), allow navigation between web pages, external resources, or sections within the same page. Key Points 1. Anchor Tags (): ○ Used to create hyperlinks. ○ The href attribute specifies the destination URL or location. 2. Types of Links: ○ External Links: Navigate to a di erent website. ○ Internal Links: Navigate within the same website. ○ Anchor Links: Jump to a specific section on the same page.
